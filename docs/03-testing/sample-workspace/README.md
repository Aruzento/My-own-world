# Sample Workspace

Это маленький пример workspace для первого знакомства с My own world.

Как использовать:

1. Откройте приложение.
2. Нажмите кнопку папки в sidebar.
3. Выберите папку `docs/03-testing/sample-workspace`.

Внутри есть стартовая карточка, карта кампании и task tracker.
