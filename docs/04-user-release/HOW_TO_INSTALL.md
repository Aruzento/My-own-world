# How To Install

Пока desktop-направление находится в стадии foundation/spike.

## Browser

Для browser-версии запустите:

```powershell
npm run dev:web
```

## Desktop

Для проверки desktop-режима используйте:

```powershell
npm run desktop:dev
```

Полная инструкция для installer будет заполнена перед первым пользовательским desktop-релизом.
