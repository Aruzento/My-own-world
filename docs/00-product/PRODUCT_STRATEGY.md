# Product Strategy

Стратегия MyOwnWorld строится вокруг превращения локальной вики в операционную систему для НРИ-мира.

## Основные опоры

1. CharacterModel: единая модель персонажей, существ, здоровья, навыков, инвентаря и эффектов.
2. Rule Tree: структурированная база правил, homebrew и справочников.
3. Knowledge Graph: связи между сущностями мира.
4. World Package System: экспорт, импорт, переиспользование и будущий workshop.

## Текущий фокус

Сначала нужно завершить организацию проекта и release handoff, затем переходить к CharacterModel. Без порядка в документации и релизных материалах новые доменные системы будут сложнее проверять и передавать.
