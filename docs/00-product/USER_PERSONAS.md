# User Personas

## Мастер кампании

Создает мир, ведет карты, управляет существами, готовит сцены, хранит правила и заметки.

## Игрок

Получает презентационный режим, видит карту, может в будущем взаимодействовать с персонажем и инвентарем через ограниченный интерфейс.

## Автор контента

Создает карточки, правила, шаблоны, пакеты миров и материалы для переиспользования.

## AI-ассистент проекта

Помогает развивать код и документацию, но обязан работать по `AGENTS.md`, читать нужные документы и оставлять понятный след изменений.
