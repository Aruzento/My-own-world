# Product Vision

MyOwnWorld - local-first рабочее место мастера НРИ для хранения мира, ведения кампании, подготовки сцен и управления игровыми сущностями.

Главная цель: сделать так, чтобы мир, правила, персонажи, карты, задачи и будущие пакеты контента жили в одной понятной системе, а не распадались на набор отдельных файлов и инструментов.

## Принципы

- Владелец мира контролирует свои данные.
- Browser и desktop версии должны работать с одним workspace-форматом.
- Новые функции должны усиливать одну из доменных опор: CharacterModel, Rule Tree, Knowledge Graph, World Package System.
- AI-разработка должна оставлять проект понятным человеку.
