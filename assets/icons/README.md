# Icons

The local SVG sprite in `rpg-ui.svg` uses original project icons drawn in a Lucide-like line style.

Reference set: Lucide Icons, ISC License.
Source: https://lucide.dev/ and https://github.com/lucide-icons/lucide/blob/main/LICENSE

Project rule: keep icons local, monochrome, `currentColor`-driven, and consistent with the fantasy/RPG UI tone.
